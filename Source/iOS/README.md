# Harvest iOS Application

## Build Instructions

### Prerequisites

- XCode 9.0 (or above)
- Swift 4.1 (or above)
- CocoaPods


### Instructions

All instructions assume your current working directory is in "COS301-Capstone/Source/iOS"

```
cd Harvest
pod install
```
